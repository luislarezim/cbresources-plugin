A powerful plugin to manage and display resources in WordPress with AJAX filtering and custom templates.

== Description ==

CB Resources Plugin is designed to help WordPress site owners manage and display resources efficiently. With AJAX filtering and customizable templates, it provides a dynamic and interactive experience for users looking to filter and access resources quickly.

== Installation ==

1. Upload the `cb-resources-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the shortcode `[cb_resources_filter]` in your pages or posts where you want the resources filter to appear.

== Usage ==

Insert the shortcode `[cb_resources_filter]` into the content area of your page or post to display the resources filter.

Example:

[cb_resources]


For detailed customization options and advanced usage, please refer to the plugin's documentation.

== Frequently Asked Questions ==

= How do I customize the resource templates? =

You can override the default templates by copying them from the plugin's templates directory to your theme's directory and then customizing them to fit your needs.

= Does this plugin support AJAX filtering? =

Yes, AJAX filtering is a key feature that provides a seamless experience for users to filter resources without reloading the page.
